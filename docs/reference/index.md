# Package index

## Main Functions

Core methods for network-based transcriptomic analysis

- [`run_tkoi()`](run_tkoi.md) : Run tKOI Analysis
- [`run_gene_enrichment()`](run_gene_enrichment.md) : Run Gene
  Enrichment and Compare with TKOI Data
- [`get_neighboring_nodes()`](get_neighboring_nodes.md) : Retrieve
  Neighboring Nodes in a Knowledge Graph
- [`make_gene_exploration_plot()`](make_gene_exploration_plot.md) :
  Create Gene Exploration Plot
- [`export_gene_exploration_data()`](export_gene_exploration_data.md) :
  Export Gene Exploration Data
- [`export_network_summary_statistics()`](export_network_summary_statistics.md)
  : Export Network Summary Statistics from tKOI Result
- [`visualize_topn()`](visualize_topn.md) : Visualize Top Network
  Enrichment Statistics
- [`plot_network()`](plot_network.md) : Plot a Local Network Around a
  Target Node from tKOI Results
- [`compute_network_enrichment()`](compute_network_enrichment.md) :
  Compute Z-Score and P-Value for Node Pagerank

## Data and Annotations

Annotation datasets used for node interpretation

- [`tkoi_net`](tkoi_net.md) : Human-Specific Heterogeneous Network
- [`anatomy_annotation`](anatomy_annotation.md) : Anatomical Entity
  Annotations from UBERON
- [`celltype_annotation`](celltype_annotation.md) : Cell Type Ontology
  Annotations (CL)
- [`clinicallab_annotation`](clinicallab_annotation.md) : Clinical
  Laboratory Measurement Annotations (LOINC)
- [`complex_annotation`](complex_annotation.md) : Protein and Molecular
  Complex Annotations
- [`compound_annotation`](compound_annotation.md) : Chemical Compound
  Annotations
- [`disease_annotation`](disease_annotation.md) : Disease Ontology (DO)
  Annotations
- [`ec_annotation`](ec_annotation.md) : Enzyme Commission (EC) Number
  Annotations
- [`go_annotation`](go_annotation.md) : Gene Ontology (GO) Term
  Annotations
- [`mirna_annotation`](mirna_annotation.md) : Human miRNA Annotations
  (miRBase-style)
- [`pathway_annotation`](pathway_annotation.md) : Pathway Annotations
  (WikiPathways)
- [`protein_annotation`](protein_annotation.md) : Protein Annotations
  (UniProt-Based)
- [`proteindomain_annotation`](proteindomain_annotation.md) : Protein
  Domain Annotations (Pfam)
- [`proteinfamily_annotation`](proteinfamily_annotation.md) : Protein
  Family Annotations (Pfam Clans)
- [`pwgroup_annotation`](pwgroup_annotation.md) : Pathway Group
  Annotations (Reactome Interaction Sets)
- [`reaction_annotation`](reaction_annotation.md) : Biochemical Reaction
  Annotations (KEGG-Based)
- [`genes`](genes.md) : Gene Metadata
- [`human_metabolites`](human_metabolites.md) : human_metabolites:
  Parsed Human Metabolite Metadata from HMDB
